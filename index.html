Historia

Lenguajes de Programacion

  • HTML (1991)

    HyperText Markup Language

    Tim Berners-Lee, creador de la World Wide Web, propuso HTML como un lenguaje de marcado para estructurar la información en documentos compartidos en la web. La versión inicial fue simple, destinada a enlaces y texto básico.

  • CSS (1996)

    Cascading Style Sheets

    Håkon Wium Lie y Bert Bos propusieron Cascading Style Sheets (CSS) como una solución para separar el contenido (HTML) de su presentación. Esta separación permitió un diseño más flexible y mantenible para las páginas web, con reglas de estilo que describen la apariencia de los elementos.

  • JS(1995 en adelante)

    Javascript

    Java, creado por Sun Microsystems, se introdujo como un lenguaje de programación versátil. En el desarrollo web, Java se utilizó inicialmente en el lado del cliente con applets, pero pronto se expandió al lado del servidor con tecnologías como JavaServer Pages (JSP) y servlets, permitiendo la creación de aplicaciones web dinámicas.

  • Evolución Continua (2000s en adelante)

    Futuro

    HTML5, introducido en 2014, trajo nuevas capacidades y mejoras semánticas a HTML. CSS continuó evolucionando con versiones más recientes para mejorar la estilización. Java, aunque ha perdido popularidad en el lado del cliente, sigue siendo relevante en el lado del servidor con frameworks como Spring y en el desarrollo de aplicaciones empresariales. La interoperabilidad entre estas tecnologías ha permitido el desarrollo web moderno y dinámico.

¿Para Qué se Utiliza el HTML?


El HTML ha sido el lenguaje por defecto utilizado para los documentos y sitios web desde la década de 1990. Funciona con los navegadores para ayudarles a entender la estructura de un sitio web y el estilo asociado a ellos.

Esencialmente, el HTML es un plano que indica al navegador web cómo armar tu sitio web para el visitante. Cuando un usuario teclea tu URL, el navegador monta la página basándose en los bloques de construcción establecidos en el código HTML.

Hoy en día, la mayoría de los desarrolladores utilizan la versión 5.3 de HTML, que es la más actualizada y que comúnmente conocemos como HTML5. Esta versión busca ofrecer elementos y atributos nuevos a los usuarios, adecuados a las necesidades de programación web modernos.

¿QUE ES CSS?

CSS es el segundo lenguaje más básico y esencial para crear páginas web. El primero sería HTML, con el que se define el contenido de la página. El segundo CSS, con el que se define la parte de la presentación, es decir, cómo deben mostrarse los elementos de la página, su posición, forma, espaciados, colores y en resumen, toda la parte estética.

¿POR QUE SE USA CSS?

La idea de CSS es la de utilizar el concepto de separación de presentación y contenido. Este concepto se basa en que, como programadores, lo ideal es separar claramente el código que escribimos. ¿Por qué? Porque con el tiempo, esto hará que el código sea más fácil de modificar y mantener.

¿QUE ES JS?

JavaScript es un lenguaje de programación de alto nivel y orientado a objetos diseñado para agregar interactividad a las páginas web. Se ejecuta en el navegador del usuario y permite la manipulación del DOM, siendo esencial en el desarrollo web tanto en el frontend como en el backend con Node.js.

    ¿POR QUE SE USA JS?

    JavaScript se emplea para hacer páginas web interactivas, manipular el DOM en tiempo real, validar formularios, crear animaciones y efectos visuales, realizar comunicación asíncrona con el servidor, desarrollar aplicaciones web completas en el lado del cliente y del servidor, y utilizar frameworks y bibliotecas como React, Angular y Vue.js para simplificar el desarrollo de aplicaciones web complejas. En esencia, JavaScript es esencial para mejorar la experiencia del usuario y la eficiencia en el desarrollo web.

HTML

    1. Etiqueta:

  • Una etiqueta HTML es un elemento fundamental del lenguaje de marcado. Está compuesta por caracteres especiales rodeados por corchetes angulares < >
  • Ejemplo: <p >,<h1 >,< div>, <a >

    1. Atributos:

  • Los atributos proporcionan información adicional sobre la etiqueta y se incluyen dentro de su apertura.
  • Están compuestos por un nombre y un valor, separados por un signo igual '(=)' y encerrados entre comillas.
  • Ejemplo: <a href="https://www.ejemplo.com">Enlace </a >. En este caso, href es el atributo y https://www.ejemplo.com es el valor.

    1. Valor:

  • En el contexto de HTML, el "valor" se refiere al contenido o la información que se encuentra entre la etiqueta de apertura y cierre.
  • Ejemplo: <p >Este es un párrafo.</p >. Aquí, "Este es un párrafo." es el valor del elemento <p > .

    1. Contenido:

  • El contenido es la información que se encuentra entre la etiqueta de apertura y la de cierre.
  • No todas las etiquetas tienen contenido. Algunas, como <img > o <br>, no requieren contenido y se cierran por sí mismas.
  • Ejemplo con contenido: <h1>¡Hola, mundo!</h1>. Aquí, "¡Hola, mundo!" es el contenido del encabezado <h1>.

    1. <head>:

  • La sección <head> de un documento HTML contiene información crucial sobre el documento que no es visible directamente en el navegador.
  • Elementos comunes dentro de <head> incluyen:
    • <meta>: Define metadatos como el conjunto de caracteres, la descripción, las palabras clave, etc.
    • <title>: Define el título de la página, que aparecerá en la pestaña del navegador o en los resultados de búsqueda.
    • <title>: Se utiliza para vincular el documento a hojas de estilo externas, iconos, etc
    • <title>: Puede incluir scripts JavaScript o referencias a archivos externos de script.
    • Otros elementos específicos según las necesidades del documento.

    1. <body>:

  • La sección <body> de un documento HTML contiene el contenido principal visible en el navegador.
  • Elementos comunes dentro de <body> incluyen:
    • Encabezados (<h1>, <h2>, ..., ): Definen encabezados de diferentes niveles.
    • Párrafos (<p>): Contienen bloques de texto.
    • Listas (<ul>, <ol>, <li>): Para crear listas no ordenadas y ordenadas.
    • Imágenes (<img>): Para mostrar imágenes.
    • Otros elementos según la necesidad del contenido.

CSS

    1. Selector:

  • El selector apunta a los elementos HTML a los que se aplicará el estilo.
  • Puede ser el nombre de un elemento HTML, una clase, un ID o incluso combinaciones más complejas.

    1. Propiedad:

  • La propiedad es el aspecto específico del estilo que deseas aplicar al elemento seleccionado.
  • Puede incluir propiedades como color,font-sizeb, margin, padding, entre otras.

    1. Valor:

  • El valor es la configuración específica que le das a la propiedad.
  • Puede ser un número, una cadena de texto, un color, una medida, etc.

JS

    1. Nombre (Identificador):

  • En JavaScript, el nombre se refiere al identificador que asignas a una función, variable u objeto.
  • Debe seguir ciertas reglas, como comenzar con una letra, un guion bajo (_) o un símbolo de dólar ($).
  • Puedes usar letras, números, guiones bajos y símbolos de dólar en los nombres.

    1. Parámetro:

  • Los parámetros son valores que se pasan a una función para que pueda realizar su tarea.
  • Se definen entre los paréntesis de la función y se utilizan como variables locales dentro de la función.

    1. Sentencia:

  • Una sentencia es una instrucción ejecutable que realiza una acción o declara algo.
  • Puede ser una asignación, una declaración de control de flujo, una llamada a función, etc.
